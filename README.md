Para começar, vá ao Visual Code, e selecione a pasta Documentos, ou qualquer pasta de sua preferência, após a seleção da pasta, vá ao terminal e insira o seguinte código:

npm i -g expo-cli

Após isso, você poderá criar um projeto da seguinte forma:

npx create-expo-app NOMEDOPROJETO 

Efetuando isso, execute o projeto com npx expo start ou npx expo start --tunnel
